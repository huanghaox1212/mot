    1.1 元素与集合¶

    注：基础知识请参考高中必修课本。

    这个东西有点难讲，因为很多内容都偏组合，而且题目很杂，非常杂。

    因此如果不满意这篇讲解请见谅。

    1.1.1 子集个数初步¶

    • n元集合的子集个数为2^n
    • n元Venn图将平面划分为2^n个区域
    • 这两点是显然的。

    注：需要注意的是，任意个集合都存在Venn图，只是超过6元后就无法用凸图形绘制。

    比如这是七元Venn图

    常见的Venn图：

    解题策略：

    • 乘法原理、加法原理

    • 容斥原理

    在此仅附上三元情形，推广的见本章后面。

    |A\cup B\cup C|=|A|+|B|+|C|-(|A\cap B|+|B\cap C|+|C\cap A|)+|A\cap B\cap C|

    该定理的证明参考Venn图。

    1.1.3 不动点集合¶

    集合\{x\in\text{dom}(f)|f(x)=x\}称为函数f的不动点集合，其中元素称为函数f的不动点。dom(f)表示f的定义域(domain)。

    • 熟悉集合\{x\in\text{dom}(f)|f(x)=x\}与\{x\in\text{dom}(f)|f(f(x))=x\}的关系（习题5）
    • 含参函数的不动点集合与参数取值范围的关系（习题5），熟练使用判别式

    注意，本节内容为入门级别。如需更深入了解请前往组合模块。

    • 构造一组能取到最值的可能的情形，即构造一组取等的充分条件。

    • 证明若存在最值，则最值不可能大于/小于自己求（cai）出的那个最值。

    1.2 容斥原理¶

    |{\large{\cup}}_i A_i|=\sum\limits_i|A_i|-\sum\limits_{1\le i<j\le n}|A_i\cap A_j|+...+(-1)^{n+1}|{\large\cap}_i A_i|

    使用归纳法证。其中{\large{\cup}}_i A_i=A_1\cup A_2\cup...\cup A_n，{\large\cap}_i A_i类似。\sum为求和符号。

    因为这涉及到集合，所以仅挂在这里，更深入的题目会在组合模块中提及。

    1.5 子集族¶

    概念：满足一定性质的一系列子集

    题型：

    • 求满足特定性质的子集的个数
    • 求满足特定性质的子集的阶的最值
    • 求最大（小）的n，使任意阶为n的子集族，满足特定性质

    • “好子集”存在的充要条件

    • 求满足特定性质的子集族的阶的最值

    • 构造特定性质的子集族
    • 求满足特定性质的子集族的阶的最大最小值
    • 构造特定性质的子集族

    • 好子集族存在的充要条件

    • 杂题
